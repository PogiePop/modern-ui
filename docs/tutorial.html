第一章：环境搭建与第一个窗口

1.1 克隆项目与构建

环境要求：Windows x64 / Visual Studio 2022 / CMake 3.14+ / OpenGL 3.3
git clone https://github.com/PogiePop/modern-ui.git
cd modern-ui
mkdir build && cd build
cmake .. -G "Visual Studio 17 2022" -A x64
cmake --build . --config Release

1.2 最小程序

创建 main.cpp

#include "core/App.hpp"
#include "widgets/Container.hpp"
#include "widgets/Label.hpp"

int main() {
    ui::App app;
    app.init("Hello ModernUI", 400, 300);

    auto label = std::make_unique<ui::Label>("你好，ModernUI！");
    label->setFont(app.getFont());

    auto root = std::make_unique<ui::Container>();
    root->addChild(std::move(label));
    app.setRoot(std::move(root));
    app.run();
}

运行后显示一个 400x300 的窗口，中央显示 "你好，ModernUI！"。

1.3 理解架构

App — 应用程序入口，管理窗口、渲染循环、事件分发

Widget — 所有控件的基类，提供位置、大小、可见性、动画等

Container — 布局容器，按水平/垂直方向排列子控件

Painter — 渲染器，将控件树绘制到屏幕

Font — 字体系统，支持 TTF/TTC 动态字形加载

Theme — 主题系统，20种颜色角色统一管理

第二章：布局系统

2.1 Container 容器

Container 是 ModernUI 的核心布局组件，相当于 CSS 的 Flexbox。

// 默认垂直排列
auto col = std::make_unique<Container>(LayoutDirection::Vertical);
col->setSpacing(8);                       // 子控件间距
col->setPadding(EdgeInsets::all(12));     // 内边距
col->setAlign(Container::Align::Center);   // 对齐方式

2.2 Row 水平布局

auto btn1 = std::make_unique<Button>("OK");

// 固定宽度 200px
row->setChildSizing(0, Sizing::fixed(200), Sizing::intrinsic());
// 弹性填充（权重 2）

第五章：主题系统

5.1 颜色角色与 Theme

Theme dark = Theme::defaultDark();
Theme light = Theme::defaultLight();

// 20 种颜色角色
dark.setColor(ColorRole::Primary, Color::fromHex(0xFF6366F1));
dark.setColor(ColorRole::Success, Color::fromHex(0xFF22C55E));
dark.setColor(ColorRole::Danger,  Color::fromHex(0xFFEF4444));

// 组件使用 setColorRole() 自动跟随主题
label->setColorRole(ColorRole::Text);
btn->setVariant(Button::Primary);     // 使用 Primary 颜色
pb->setColorRole(ColorRole::Success);

5.2 主题切换动画

ThemeTransition transition;
transition.setImmediate(dark);            // 立即应用
transition.setTarget(light, 0.5f);       // 0.5秒平滑过渡

// 每帧驱动
struct ThemeDriver : Widget {
    void tickAnimation(float dt) override { transition.tick(dt); }
    Size measure(const Size&) const override { return {0,0}; }
    void paint(Painter&) override {}
};

5.3 全局配置 Config

auto& cfg = Config::instance();
cfg.darkMode = false;                // 亮色主题
cfg.multiThreaded = true;            // 多线程布局
cfg.scrollStep = 50.0f;              // 滚轮灵敏度
cfg.themeTransitionDuration = 0.5f;  // 动画时长
cfg.loadFromFile("app.cfg");
cfg.saveToFile("app.cfg");

第七章：发布部署

7.1 CMake Install 打包

cd build
cmake --install . --prefix ./install --config Release
# 输出:
# install/include/modernui/  — 头文件
# install/lib/               — modernui_static.lib + modernui_shared.lib
# install/bin/               — modernui_shared.dll + glfw3.dll

7.2 静态库与动态库

target_link_libraries(your_app PRIVATE modernui_static glfw3 glad opengl32)
