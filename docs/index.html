ModernUI

C++20 现代 GPU 加速 GUI 工具包。40+ 组件，主题切换动画，CJK 动态字体，右键菜单，全部现代化内置样式。

快速开始 组件预览 GitHub

特性

ModernUI 提供完整的 GUI 开发体验

🎨 40+ 组件

Button, Label, TextInput, Dropdown, TabView, Table, DatePicker, BarChart 等全部常用组件

🌓 主题动画

暗色/亮色 0.4s 平滑过渡，ColorRole/FontSize/Spacing/BorderRadius 全插值

🔤 CJK 字体

Unicode 按需光栅化，中文/日文/韩文/Emoji 动态加载，多图集页

🖱 右键菜单

TextInput/TextEdit/Label 支持复制/粘贴/撤销/重做/全选

⚡ 多线程

可选 ThreadPool 并行 measure() 布局计算

🎯 图标系统

45 种内置几何矢量图标，所有组件可设置

快速开始

mkdir build && cd build
cmake .. -G "Visual Studio 17 2022" -A x64
cmake --build . --config Release
./Release/exp.exe
