组件预览

ModernUI 提供 40+ 内置组件，每个组件支持多种现代化样式变体。

基础组件

Button

5 种变体 (Primary/Secondary/Danger/Success/Ghost)，3 种形状 (Default/Pill/Compact)

setVariant() setPill() setCompact() setCornerRadius()

Label

支持主题色角色、对齐、选中复制、右键菜单

setColorRole() setTextAlign() setFontSize()

TextInput

单行输入，选中/拖拽/右键菜单/撤销重做/CJK 输入

setPlaceholder() setOnTextChanged() addContextMenuItem()

TextEdit

多行编辑器，UTF-8 码点光标，换行折叠

setText() setFont() selectAll() undo() redo()

Switch

3 种尺寸 (S/M/L)，支持主题色变体

setChecked() setColorRole() setMinSize()

Checkbox

3 种尺寸，5 种主题色，圆角复选框

setChecked() setBoxSize() setColorRole()

RadioButton

RadioGroup 分组，5 种主题色，3 种尺寸

setSelected() setDotSize() setColorRole()

Slider

可自定义拇指大小、轨道高度、主题色

setValue() setStep() setColorRole() setThumbSize()

ProgressBar

5 种颜色变体，圆角填充条

setValue() setColorRole()

LoadingSpinner

8 段旋转动画，支持主题色和尺寸变化

setColorRole() setSpeed() setSpinnerSize()

容器组件

Container

Row/Column 布局，FlexBox 对齐，测量缓存

setDirection() setPadding() setSpacing() setAlign() setChildSizing()

ScrollArea

滚动容器，左侧滚动条，动态显示/隐藏

content() scrollbarWidth() setScrollY()

TabView

多标签页，可选内容边框，主题色 tab 栏

addTab() setCurrentTab() setContentBorder()

Card

4 种海拔高度，圆角阴影卡片

setElevation() setBorderRadius()

Border

3 种线型 (Solid/Dashed/Dotted)，主题色

setStroke() setBorderStyle() setCornerRadius()

CollapsePanel

3 种变体 (Default/Bordered/Card)，展开/折叠

setExpanded() setVariant() setContent()

Form / FormField

2 种标签位置 (Top/Left)，紧凑模式

addField() setCompact()

选择与输入

Dropdown

下拉选择，方向自适应 (Down/Up/Auto)，清除 X

addItem() setSelectedIndex() clearSelection()

Select

内联列表选择，鼠标悬停高亮

addItem() setSelectedIndex() setOnChanged()

InputNumber

数字输入，+/- 按钮，步进，范围限制

setRange() setStep() setValue()

DatePicker

日历选择器，月/年导航，主题色

setDate() setOnDateChanged()

数据展示

Table

数据表格，交替行色，可调行高/表头高

setColumns() addRow() setRowHeight()

Pagination

分页导航，圆角/方角，主题色

setTotalPages() setCurrentPage() setRounded()

StatCard

统计卡片，6 色变体 + 3 尺寸 + 趋势箭头

setLabel() setValue() setTrend() setVariant()

BarChart

柱状图，主题色，数据标签

setData() setColorRole()

弹窗与消息

Dialog

模态对话框，拖动标题栏，ESC 关闭

setTitle() show() close() setDialogSize()

MessageBox

消息框，自定义按钮，5 种图标类型

setTitle() setMessage() setIcon() addButton()

ContextMenu

右键菜单，内置编辑项 + 自定义项

addItem() addSeparator() show()

Toast

Toast 通知，滑入动画，图标 + 消息

showToast(msg, icon, duration)

Tooltip

5 种位置 (Top/Bottom/Left/Right/Auto)，悬停0.75s触发

setTooltip(widget, text)

MessageBubble

聊天气泡，3 种变体 (Default/Compact/Card)

setSender() setText() setSide() setVariant()

装饰与导航

Badge

6 色徽章：Primary/Success/Danger/Warning/Info/Default

setVariant() setText()

Avatar

头像，首字母提取，主题色

setName() setSize() setColorRole()

Image

图片显示，4 种缩放模式 + 圆角 + 阴影

setTexture() setScaleMode() setImageRadius()

Link

超链接，主题色 + 下划线样式

setUrl() setColorRole() setUnderline()

NavMenu

导航菜单，嵌套层级，图标支持

setItems() MenuItem{label, icon, action}

Spacer / Divider

弹性空间 + 分割线 (Solid/Dashed/Dotted)

Spacer(size) Divider(dir, style)

图标系统 (45种)

导航

ArrowUp/Down/Left/Right, ChevronUp/Down

操作

Check, Close, Plus, Minus, CheckCircle, CloseCircle

状态

Info, Warning, Error, Question, Success

对象

Search, User, Settings, Home, Heart, Star, Mail, Lock

媒体

Play, Pause, Stop, Refresh, Volume, Mute, Clock

文件

File, Folder, Download, Upload, Image, Copy, Edit

UI

Menu, More, External, Filter, Grid, List, Eye, EyeOff, Bell, Calendar, Tag
