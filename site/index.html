Architecture

main.cpp
src/core/   — App, Window, Event, Painter, QuadBatcher, Animation, Icon
src/res/    — Shader, Texture, Font, Theme, StyleSheet, BlurShader
src/widgets/ — Widget base + 50+ component implementations
src/utils/  — HttpClient, Navigation, AsyncTask

Layer Dependencies

widgets → res + core → core → GLFW/glad/GLM/stb

Key Systems

  • Widget tree: unique_ptr ownership, virtual measure/layout/paint/hitTest
  • Rendering: batched VBO with auto-flush on texture change, SDF rounded rects
  • Events: target+bubble dispatch, pointer capture for drag, screenRect coordinates
  • Layout: two-pass (measure→layout), Fixed/Fill/Intrinsic sizing, Flex/Grid/Responsive
  • Theme: enum tokens→fixed arrays, light/dark presets, CSS stylesheet parser
  • Animation: per-widget opacity/slide, 9 easing functions, 60fps scheduler

Customize & Extend

Creating a Custom Widget

class MyWidget : public ui::Widget {
public:
    const char* typeName() const override { return "MyWidget"; }
    Size measure(const Size& avail) const override { return {100, 30}; }
    void paint(Painter& painter) override {
        Rect r = screenRect();
        painter.drawRoundedRect(r, Color{0.2f,0.5f,0.9f,1}, 8);
        painter.drawText(r, "Hello", Color{1,1,1,1}, TextAlign::Center);
    }
};

Adding to a Container

auto root = std::make_unique<ui::Container>(ui::LayoutDirection::Vertical);
root->addChild(std::make_unique<MyWidget>());

Custom Styling

widget->setShadow({2,3}, 8, {0,0,0,0.35f});
widget->setGradient({1,0.2f,0.2f,1}, {0.2f,0.2f,1,1});
widget->setCornerRadius(12);
widget->setFrosted(true);
widget->setZIndex(100);

Theme Customization

ui::Theme theme = ui::Theme::defaultDark();
theme.setColor(ui::ColorRole::Primary, Color{0.2f,0.6f,1,1});
theme.setFontSize(ui::FontSizeRole::Body, 16);

Getting Started

git clone https://github.com/PogiePop/modern-ui.git
cd modern-ui/build
cmake .. -G "Visual Studio 17 2022" -A x64
cmake --build .
cp ../external/glfw/lib-vc2022/glfw3.dll Debug/
./Debug/exp.exe
